# 🤖 Jarvis AI Assistant

A beginner-friendly AI Voice Assistant built with Python.

This project was built from scratch while learning Python and AI fundamentals.

---

## ✨ Features

* 🎤 Voice Input
* 🧠 Speech to Text using Faster Whisper
* 🗣️ Text to Speech
* 👋 Wake Word Detection
* 🌐 Open Google
* ▶️ Open YouTube
* 💻 Open GitHub
* 🤖 Open ChatGPT
* 🖩 Open Calculator
* 📝 Open Notepad
* ⏰ Tell Current Time
* 📅 Tell Today's Date
* 🔄 Continuous Listening
* 🚪 Exit Command

---

## 🛠️ Technologies Used

* Python
* Faster Whisper
* SoundDevice
* SciPy
* pyttsx3
* Edge TTS
* Playsound3

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Jarvis-AI.git
```

### 2. Move into the project folder

```bash
cd Jarvis-AI
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the project

```bash
python main.py
```
